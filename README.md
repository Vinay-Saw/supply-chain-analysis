# Supply Chain Analysis - OptimalFlow Logistics

## Project Overview
This repository contains the correlation analysis of supplier performance metrics for OptimalFlow Logistics. The analysis aims to identify relationships between lead times, inventory levels, order frequency, and costs to optimize procurement strategies for a major automotive manufacturer.

## Files
- `correlation.csv`: The calculated correlation coefficients between supply chain variables.
- `heatmap.png`: Visual representation of the correlation matrix using a Red-White-Green color scale.
- `q-excel-correlation-heatmap.xlsx`: Source Excel file containing the raw data and analysis.

## Key Findings
- **Cost Drivers**: There is a strong positive correlation between Supplier Lead Time and Cost Per Unit, suggesting that slower suppliers are also more expensive.
- **Performance Impact**: Delivery Performance is negatively correlated with Cost Per Unit.
